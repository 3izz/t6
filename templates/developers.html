عن المشروع

Link Library هو تطبيق سحابي مفتوح المصدر يسمح للمستخدمين بإدارة روابطهم بشكل آمن وفعال. المشروع مبني باستخدام:

  • Flask (Python) للواجهة الخلفية
  • Bootstrap للواجهة الأمامية
  • SQLite/PostgreSQL للقاعدة البيانات

كيفية المساهمة

نرحب بمساهماتكم في تطوير المشروع! يمكنك المساهمة من خلال:

  1. تطوير ميزات جديدة
  2. إصلاح الأخطاء
  3. تحسين الوثائق
  4. تحسين واجهة المستخدم

خطوات البدء

1. قم بعمل Fork للمشروع
2. قم بإنشاء فرع جديد (git checkout -b feature/amazing-feature)
3. قم بعمل Commit للتغييرات (git commit -m 'Add amazing feature')
4. قم بعمل Push للفرع (git push origin feature/amazing-feature)
5. قم بفتح طلب Pull Request

المتطلبات

لتشغيل المشروع محلياً، تحتاج إلى:

  • Python 3.7 أو أحدث
  • pip (مدير حزم Python)
